# 🚀 Multi-Tool Intelligence Dashboard (Streamlit)

Este repositorio es la culminación de mi especialización en **Data Science & Machine Learning**, integrando tres proyectos avanzados en una única interfaz interactiva desarrollada con **Streamlit**. 

Aquí demuestro mi capacidad para manejar el ciclo completo del dato: desde la extracción y limpieza hasta el modelado predictivo y la visualización profesional.

## 🛠️ Los 3 Proyectos Core:

### 1. 🎮 Recomendador de Videojuegos (ML No Supervisado)
* **Tecnología:** K-Means Clustering & NLP (Similitud de Coseno).
* **Función:** Algoritmo que analiza mecánicas y géneros para recomendar títulos similares. Incluye visualización 3D interactiva de clusters con Plotly.

### 2. 💻 Dashboard de Mercado de Hardware (Data Analysis)
* **Tecnología:** Pandas, APIs de precios y Power BI integration.
* **Función:** Análisis comparativo de la evolución de precios de GPUs/Consolas vs. Inflación, enfocado en Business Intelligence.

### 3. 🎓 Predictor de Éxito Académico (ML Supervisado)
* **Tecnología:** Scikit-Learn (Random Forest / Linear Regression).
* **Función:** Modelo predictivo que estima el rendimiento de estudiantes basándose en variables socio-educativas y hábitos de estudio.
